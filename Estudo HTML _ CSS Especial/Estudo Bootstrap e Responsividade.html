Versão do Bootstrap


Verificar no site do Bootstrap qual a versão mais atualizada para realização do projeto.

Neste curso iremos utilizar a Versão: v4



Configurando a Utilização


1 - Realizar o Download no site do Bootstrap, que está mencionada acima.

2 - Descompactar a Pasta e Incluir Dentro da Pasta do Projeto.

3 - Copiar o exemplo de Documento HTML Padrão ( Starter Template) na Aba = "Getting Starded - Introduction".

4 - Refenciar a Folha de Estilo de CSS "stylesheet" do Bootstrap da Pasta do Projeto.

" ../bootstrap/css/bootstrap.min.css "

OBS.: Ao carregar o Bootstrap no próprio diretório do projeto ao inves de um Link CDN do Site Oficial, dever excluido os atributos "INTEGRITY" e "CROSSORIGIN" [ Pois são chaves Criptográficas para Carragamento do Arquivos Externo ].

OBS.: Fazer o mesmo procedimento citado acima para os arquivos JS.
