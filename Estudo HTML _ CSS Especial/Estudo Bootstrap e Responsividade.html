Versão do Bootstrap


Verificar no site do Bootstrap qual a versão mais atualizada para realização do projeto.

Neste curso iremos utilizar a Versão: v4



Configurando a Utilização


1 - Realizar o Download no site do Bootstrap, que está mencionada acima.

2 - Descompactar a Pasta e Incluir Dentro da Pasta do Projeto.

3 - Copiar o exemplo de Documento HTML Padrão ( Starter Template) na Aba = "Getting Starded - Introduction".

4 - Refenciar a Folha de Estilo de CSS "stylesheet" do Bootstrap da Pasta do Projeto.

" ../bootstrap/css/bootstrap.min.css "

OBS.: Ao carregar o Bootstrap no próprio diretório do projeto ao inves de um Link CDN do Site Oficial, dever excluido os atributos "INTEGRITY" e "CROSSORIGIN" [ Pois são chaves Criptográficas para Carragamento do Arquivos Externo ].

OBS.: Fazer o mesmo procedimento citado acima para os arquivos JS.

Media-Queries


Definição de Estilos Diferentes para Determinadas Telas Pré Selecionadas

Tipos de mídias:
all – todos os dispositivos
aural – sintetizadores de voz
braille – leitores de Braille
embossed – impressoras de Braille
handheld – dispositivos de mão. Por exemplo: celulares com telas pequenas.
print – impressoras convencionais
projection – apresentações de slides
screen – monitores coloridas
tty – teleimpressores e terminais
tv – televisores

Exemplo de utilização:

* link rel="stylesheet" media="print" href="print.css" *


Exemplos de resoluções de telas:
320 pixels – Smartphones no modo retrato.
480 pixels – Smartphones no modo paisagem.
600 pixels – Tablets pequenos. Ex: Amazon Kindle (600×800)
768 pixels – Tablets maiores em modo retrato. Ex: iPad (768×1024)
1024 pixels – Tablets maiores em modo paisagem, monitores antigos.
1200 pixels – Monitores wide.

Links de media queries no Bootstrap:
Bootstrap Overview

Exemplos de utilização media queries:
@media (min-width: 576px) { ... }
